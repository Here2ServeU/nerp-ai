# AI Foundations

Artificial intelligence is a broad field focused on building systems that perform tasks normally associated with human reasoning, pattern recognition, or decision-making.

In this project, AI is presented in a practical way:

- Inputs: telemetry and operational data.
- Processing: model training and prediction.
- Outputs: risk labels and anomaly signals.
- Actions: operational decisions.

## Key Terms

- Feature: an input variable used by a model, such as CPU or memory.
- Label: the known outcome used for training, such as failure risk.
- Model: a mathematical system that learns from historical examples.
- Inference: using a trained model to make a prediction on new data.

## What This Repo Focuses On

This repository focuses on applied AI for operations, not theory-heavy AI. The emphasis is on using small datasets and readable Python code to demonstrate how AI supports reliability workflows.
